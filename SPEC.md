# 内容中台系统 - 产品原型设计规范

## 1. 产品定位与设计理念

内容中台系统是面向政务和企业级用户的内容管理平台，集成内容创作、审核、发布、运营分析于一体。设计风格融合政务科技感与腾讯云控制台美学，追求简洁、高级、低饱和的视觉体验。

## 2. 设计规范

### 色彩系统
- **主背景色**: #030714 (深空蓝黑)
- **卡片背景**: #0a1628 (深海蓝)
- **边框色**: #1a2744 (暗夜蓝)
- **主色调**: #3b82f6 (科技蓝)
- **辅助色**: #06b6d4 (青色)
- **成功色**: #10b981 (翠绿)
- **警告色**: #f59e0b (琥珀)
- **危险色**: #ef4444 (朱红)
- **文字主色**: #f1f5f9 (银白)
- **文字次色**: #94a3b8 (灰蓝)
- **文字弱色**: #475569 (暗灰)

### 字体系统
- **主字体**: "PingFang SC", "Microsoft YaHei", "Helvetica Neue", sans-serif
- **等宽字体**: "JetBrains Mono", "Fira Code", monospace
- **标题字号**: 24px/20px/16px/14px
- **正文字号**: 14px/13px/12px

### 间距系统
- **基础单位**: 4px
- **卡片内边距**: 24px
- **元素间距**: 16px/12px/8px
- **页面边距**: 24px

### 圆角系统
- **卡片圆角**: 12px
- **按钮圆角**: 8px
- **输入框圆角**: 6px

### 阴影系统
- **卡片阴影**: 0 4px 24px rgba(0, 0, 0, 0.4)
- **弹窗阴影**: 0 8px 40px rgba(0, 0, 0, 0.6)
- **悬浮阴影**: 0 8px 32px rgba(59, 130, 246, 0.15)

## 4. 功能模块规划

### 4.1 工作台 Dashboard
- 今日数据概览卡片
- 内容发布趋势图表
- 待审核内容列表
- 快捷操作入口
- 近期活动时间线

### 4.2 内容中心
- **内容管理**: 文章列表、搜索筛选、内容编辑
- **分类管理**: 树形分类、拖拽排序
- **标签管理**: 标签列表、标签引用统计
- **审核中心**: 待审核队列、审核历史
- **发布管理**: 发布排期、发布记录

### 4.3 门户运营
- **首页装修**: 可视化拖拽装修
- **Banner管理**: Banner列表、轮播配置
- **栏目管理**: 栏目结构、栏目配置
- **专题管理**: 专题创建、专题页面

### 4.4 资讯公告
- **新闻资讯**: 资讯列表、分类浏览
- **公告通知**: 公告列表、公告详情
- **政策文件**: 文件列表、文件预览

### 4.5 帮助中心
- **FAQ**: 常见问题分类展示
- **文档中心**: 操作手册、API文档
- **使用指南**: 视频教程、操作指南

### 4.6 数据产品
- **数据概览**: 内容数据统计
- **内容分析**: 阅读量、点赞量分析
- **用户分析**: 用户行为分析
- **报表中心**: 数据报表导出

### 4.7 资源中心
- **图片管理**: 图片上传、图库管理
- **文件管理**: 文档管理、文件共享
- **视频管理**: 视频管理、播放统计

### 4.8 系统管理
- **用户管理**: 用户列表、角色配置
- **权限管理**: 权限组、权限分配
- **系统设置**: 基本配置、日志审计

## 6. 组件规范

### 卡片组件
- 背景: #0a1628
- 边框: 1px solid #1a2744
- 圆角: 12px
- 内边距: 24px
- 标题区: 16px 标题 + 操作按钮

### 按钮组件
- **主要按钮**: 背景 #3b82f6, 白色文字
- **次要按钮**: 透明背景, #3b82f6 边框和文字
- **危险按钮**: 背景 #ef4444, 白色文字
- **禁用状态**: opacity 0.5, cursor not-allowed

### 表格组件
- 表头: #0a1628 背景
- 行悬浮: #1a2744 背景
- 边框: 1px solid #1a2744
- 斑马纹: 奇数行 #0a1628, 偶数行透明

### 表单组件
- 输入框: #0a1628 背景, #1a2744 边框
- 聚焦: #3b82f6 边框
- 标签: #94a3b8 颜色, 14px 大小

### 弹窗组件
- 遮罩: rgba(0, 0, 0, 0.7)
- 弹窗体: #0a1628 背景, #1a2744 边框
- 圆角: 16px
- 最大宽度: 600px

## 7. 图标与图片

- 图标库: FontAwesome 6.x
- 图片: Unsplash (模糊化处理)
- 装饰: CSS 渐变 + 光晕效果

## 8. 交互规范

- 页面切换: 平滑过渡, 300ms
- 按钮悬浮: scale(1.02), 阴影加深
- 菜单展开: 高度动画, 200ms
- 弹窗弹出: 缩放 + 淡入, 250ms
- 提示信息: 从右侧滑入, 300ms
