# 面包屑导航统一优化

## 调整目标

当前系统所有页面面包屑统一显示：

首页 / 一级菜单 / 当前页面

例如：

首页 / 活动运营 / 活动管理

首页 / 运营配置 / 活动分类

首页 / 系统管理 / 用户管理

存在问题：

首页并不是这些页面的父级菜单。

左侧导航已经提供首页入口。

面包屑中的首页属于冗余层级。

需要统一优化。

---

# 调整原则

删除所有面包屑中的：

首页

保留：

一级菜单 → 当前页面

结构。

# 首页特殊处理

首页（活动运营驾驶舱）

不显示面包屑。

不要显示：

首页

不要显示：

首页 / 首页

直接隐藏面包屑区域。

# 实现要求

统一封装 Breadcrumb 组件。

不要每个页面单独写死。

根据：

当前一级菜单

当前页面名称

自动生成面包屑。

# 验收标准

✓ 删除所有面包屑中的首页

✓ 首页不显示面包屑

✓ 一级菜单页面显示正确

✓ 二级菜单页面显示正确

✓ 创建活动页面显示三级面包屑

✓ 编辑活动页面显示三级面包屑

✓ 查看活动页面显示三级面包屑

✓ 所有页面风格统一

✓ 保持现有UI风格不变
