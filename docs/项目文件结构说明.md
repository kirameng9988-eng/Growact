# 内容中台系统 - 文件结构说明

## 系统概述

**系统名称**: 内容中台系统 / 天机智信运营平台

**定位**: 活动运营中心 - 属于天机智信下的业务模块

---

## 整体目录结构

```
97_2content/
├── index.html                      # 入口页（重定向到 dashboard.html）
├── dashboard.html                  # 首页/工作台
├── _floating-demo-links.html       # 浮动演示链接（辅助文件）
│
├── activity-*.html                 # 活动管理模块（主平台）
├── content-*.html                  # 内容管理模块
├── portal-*.html                   # 门户管理模块
├── news-*.html                     # 新闻资讯模块
├── data-*.html                     # 数据分析模块
├── resource-*.html                 # 资源中心模块
├── system-*.html                   # 系统管理模块
├── help-*.html                     # 帮助中心模块
│
├── zone-config.html                # 专区配置
├── hot-activity-config.html        # 热门活动配置
│
├── activity-zone/                  # H5活动专区（PC端）
├── mobimobile-Activity-zone/       # 移动端活动专区
├── h5-checkin/                     # H5活动签到
│
└── docs/                           # 文档目录
    ├── 功能菜单优化                 # 菜单结构规划
    ├── 活动发布页面功能完善规划.md
    └── ...其他规划文档
```

## 技术栈

- **CSS框架**: Tailwind CSS (CDN)
- **图标库**: FontAwesome 6.x (CDN)
- **图表库**: Chart.js (CDN)
- **字体**: PingFang SC / Microsoft YaHei / Inter
- **类型**: 纯静态HTML原型（无构建系统）
