# 活动运营中心开发规划

## 一、模块概述

在现有内容中台系统中新增「活动运营中心」模块，作为独立一级菜单，提供完整的活动管理能力。

## 二、菜单结构

```
活动运营（新增一级菜单）
├── 活动管理 (activity-list.html)
├── 活动分类 (activity-category.html)
├── 报名管理 (activity-registration.html)
├── 活动审核 (activity-review.html)
└── 活动统计 (activity-stats.html)
```

## 三、页面规划

### 3.1 活动管理 (activity-list.html) - 重点页面

**功能：**
- 表格展示所有活动
- 支持搜索（活动名称）
- 支持分类筛选、状态筛选
- 批量操作

**表格字段：**
| 字段 | 说明 |
|------|------|
| 活动名称 | 活动标题 |
| 活动分类 | 下拉分类 |
| 状态 | 草稿/待审核/已发布/已结束/已下线 |
| 报名人数 | 数字 |
| 活动时间 | 开始-结束时间 |
| 发布时间 | 发布时间 |
| 创建人 | 创建者名称 |
| 操作 | 查看/编辑/删除/提交审核/发布/下线 |

**状态流转：**
- 草稿 → 待审核 → 已发布 → 已结束
- 已发布 → 已下线

**弹窗/功能：**
- 新建活动（右侧抽屉，3个Tab）
- 查看详情弹窗
- 编辑活动抽屉
- 删除确认弹窗

**Mock数据：**
至少10条活动数据，涵盖不同状态

### 3.2 新建/编辑活动抽屉

**结构：**
右侧抽屉式，包含3个Tab

**Tab1 - 基础信息：**
- 活动名称（文本框）
- 活动分类（下拉选择）
- 标签（多选，复用标签中心）
- 活动封面（资源中心选择）
- 活动简介（文本域）
- 活动地点（文本框）
- 主办单位（文本框）
- 最大报名人数（数字）

**时间字段：**
- 活动开始时间
- 活动结束时间
- 报名开始时间
- 报名截止时间

**发布形式：**
- 草稿
- 发布

**Tab2 - 活动详情：**
- 富文本编辑器
- 工具栏含【插入资源】按钮

**Tab3 - 资源引用：**
- 复用资源中心
- Tab分类：图片/文件/视频
- 支持搜索、选择、已选展示、删除、顺序调整

**功能：**

- 详情弹窗

## 四、技术实现

### 4.1 布局
- 复用现有顶部导航栏 + 左侧菜单栏
- 内容区域使用 `ml-[220px] mt-16 p-6`
- 保持现有配色方案

### 4.2 样式
- Tailwind CSS
- FontAwesome 6.x
- 圆角、阴影保持一致

### 4.3 交互
- 所有按钮可点击
- Toast提示保存成功
- 弹窗使用固定定位遮罩层

### 4.4 数据
- LocalStorage存储Mock数据
- 假接口假保存
- 无需后端
