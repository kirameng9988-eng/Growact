## 三、页面模块

| 模块 | 位置 | 功能 |
|------|------|------|
| **统计卡片** | 顶部左侧 | 今日发布、本周发布、待发布、已下线（点击筛选） |
| **快捷操作区** | 顶部右侧 | 今日发布数、紧急待处理、即将过期 |
| **日历视图** | 中部左侧 65% | 月份切换、日期点击、状态颜色、图例 |
| **待发布队列** | 中部右侧 35% | 按时间排序、快捷操作、批量发布 |
| **日期详情弹窗** | 点击触发 | 显示当天所有发布内容及操作 |

---

## 四、待发布队列说明

右侧队列按时间升序排列，显示：
- 计划发布时间
- 内容标题
- 当前状态标签
- 快捷操作按钮

支持勾选后批量发布。

## 六、功能清单

### 6.1 主页功能 (content-publish.html)

| 功能 | 说明 |
|------|------|
| 统计卡片 | 今日发布、本周发布、待发布、已下线（可点击筛选） |
| 快捷操作区 | 今日发布数、紧急待处理、即将过期 |
| 日历视图 | 月份切换、日期点击、状态颜色、图例 |
| 待发布队列 | 按时间排序、快捷操作、批量发布 |
| 创建计划 | 打开创建发布计划弹窗 |
| 发布记录 | 跳转至发布记录页 |
| 日期详情弹窗 | 点击日历日期弹出当天内容列表 |

### 6.2 发布记录页功能 (content-publish-list.html)

| 功能 | 说明 |
|------|------|
| 搜索 | 按标题搜索 |
| 内容类型筛选 | 新闻/公告/政策/专题/全部 |
| 状态筛选 | 待发布/发布中/已发布/已下线/全部 |
| 频道筛选 | PC/移动端/小程序/全部 |
| 日期范围 | 自定义日期筛选 |
| 导出 | 导出 Excel |
| 批量操作 | 批量发布/下线/取消 |
| 分页 | 16条/页 |
| 操作 | 下线/重新发布/详情/编辑/删除 |

## 七、状态说明

| 状态 | 颜色 | 说明 |
|------|------|------|
| 待发布 | 蓝色 #3b82f6 | 定时发布，内容尚未上线 |
| 发布中 | 靛蓝 #6366f1 | 正在发布过程中 |
| 已发布 | 绿色 #10b981 | 内容已上线展示 |
| 已下线 | 灰色 #6b7280 | 内容已下架 |
| 已撤回 | 橙色 #f59e0b | 内容被撤回 |
