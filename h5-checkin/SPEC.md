# H5移动签到页 - 规格文档

## 1. 项目概述

- **项目名称**: 活动签到 H5 页面
- **类型**: 移动端 H5 单页面应用
- **核心功能**: 微信扫码签到流程演示
- **目标用户**: 活动参与者
- **技术栈**: HTML + Tailwind CSS + 原生 JavaScript

## 2. 设计规范

### 2.1 色彩系统
- **主色**: #1d4ed8 (蓝色)
- **成功色**: #10b981 (绿色)
- **警告色**: #f59e0b (橙黄)
- **错误色**: #ef4444 (红色)
- **背景色**: #f8fafc (浅灰白)
- **卡片背景**: #ffffff (纯白)
- **文字主色**: #1e293b (深灰)
- **文字次色**: #64748b (中灰)

### 2.2 字体
- **主字体**: -apple-system, BlinkMacSystemFont, "PingFang SC", "Helvetica Neue", sans-serif
- **标题**: 20px, font-weight: 600
- **正文**: 16px, font-weight: 400
- **辅助**: 14px, font-weight: 400

### 2.3 间距
- **页面内边距**: 16px
- **卡片内边距**: 20px
- **元素间距**: 12px / 16px / 24px

### 2.4 圆角
- **卡片**: 16px
- **按钮**: 12px
- **图片**: 12px

### 2.5 阴影
- **卡片阴影**: 0 4px 20px rgba(0, 0, 0, 0.08)
- **按钮阴影**: 0 4px 12px rgba(29, 78, 216, 0.3)

## 3. 页面状态

| 状态 | 说明 | 触发条件 |
|------|------|----------|
| loading | 加载中 | 页面初启 |
| pending | 待签到 | 默认状态 / ?status=pending |
| success | 签到成功 | 点击确认签到 / ?status=success |
| signed | 已签到 | 重复扫码 / ?status=signed |
| expired | 签到失效 | 活动结束 / ?status=expired |
| loggedout | 未登录 | 无认证信息 / ?status=loggedout |

## 4. 页面结构

### 4.1 顶部导航栏
- 高度: 56px
- 左侧: 返回按钮 (← 返回)
- 中间: 标题 (活动签到)
- 背景: 白色，轻底部阴影

### 4.2 内容区域
- flex-1, 可滚动
- 安全区域适配 (env safe-area-inset-*)

### 4.3 底部固定按钮区
- 固定底部
- 内边距: 16px + 安全区
- 按钮高度: 48px
- 渐变背景 (主色 → 略浅)

## 5. 各状态页面

### 5.1 加载中状态
- 全屏居中
- Logo动画
- 文案: "正在获取签到信息..."
- 时长: 1.5秒自动切换

### 5.2 待签到状态
- 活动卡片 (封面图 + 信息)
- 用户信息展示 (姓名 + 公司 + 手机)
- 黄色警告提示
- 底部确认签到按钮

### 5.3 签到成功状态
- 大绿色勾选图标
- 成功文案
- 签到信息卡片 (活动名 + 时间 + 地点 + 用户)
- 底部返回按钮

### 5.4 已签到状态
- 大成功图标
- 文案: "您已完成签到"
- 显示签到时间
- 提示: "无需重复签到"
- 底部返回按钮

### 5.5 签到失效状态
- 大警告图标
- 文案: "签到已结束"
- 副文案: "当前活动签到时间已截止"
- 底部查看活动详情按钮

### 5.6 未登录状态
- 提示: "请先登录后签到"
- 统一身份认证登录按钮

## 7. 交互规范

- 按钮点击: scale(0.98) + 颜色加深
- 状态切换: opacity fade 300ms
- 成功动画: 勾选图标 scale 弹跳
- 页面切换: slide-up 效果

## 8. URL参数

- `?status=pending` - 待签到
- `?status=success` - 签到成功
- `?status=signed` - 已签到
- `?status=expired` - 签到失效
- `?status=loggedout` - 未登录
