## 1. 设计价值观与原则

> **天机智信规范** 秉承包容、多元、进化和连接的设计价值观,引导文字系统制定"好用"、"好记"和"美观"的字体设计原则。

字体设计原则:

- **有规律和韵律** — 字号梯度统一、行高节奏一致
- **像素对齐** — 文字渲染清晰、避免半像素模糊
- **层次清晰** — 通过字号、字重、颜色拉开信息层级
- **大小对比和谐** — 不同场景的字号对比既明显又克制

## 3. 字体系统 (Typography)

### 3.1 字体家族

| 平台 / 语言 | 推荐字体 |
| --- | --- |
| macOS · 简体中文 | `PingFang SC` |
| macOS · 英文 / 数字 | `PingFang HK` |
| Windows · 简体中文 | `Microsoft YaHei`(微软雅黑) |
| Windows · 英文 | `Segoe UI` |
| 数字 / 数据 | `DIN` / `TCloudNumber`(等距数字,缩放时粗细稳定) |

**Font-family 优先表(可直接复制到 CSS):**

```css
font-family: -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto,
             "Hiragino Sans GB", "Microsoft YaHei UI", "Microsoft YaHei",
             "Source Han Sans CN", sans-serif,
             "Apple Color Emoji", "Segoe UI Emoji", "Segoe UI Symbol";
```

> 浏览器渲染的是"默认字体"而非一定是"系统字体"。Windows 7 浏览器默认渲染中易宋体 (SimSun),而非微软雅黑,实际开发需要注意。
>
> **数字字体**:在展示表格数据时,应使用等距的表格数字(`TCloudNumber`),相比普通字体的数字能实现更好的对齐,便于快速扫描对比;并能满足在缩放中保持稳定可见的粗细。

### 3.2 字阶 (Type Scale)

#### 第一字阶 (基础组件,步长 2px)

| 名称 | 字号 | 行高 | 用途 |
| --- | --- | --- | --- |
| 移动端最小 | 10 | 14 | 移动端最小可用 |
| 提示或标签 | 12 | 16.8 | 辅助标签、徽标 |
| 辅助内容 | 14 | 19.6 | Form label、辅助文本 |
| 主要内容 (Body) | 14 | 22 | 默认正文、表单值 |
| 主要内容 (Body) | 16 | 22.4 | 主要正文 |
| 内容标题 | 18 | 25.2 | 卡片小标题 |
| banner 内容 | 20 | 28 | 段落标题 |
| banner 内容 | 24 | 33.6 | 二级标题 |
| 标题-卡片 | 28 | 39.2 | 卡片大标题 |
| 大标题 | 36 | 50.4 | 模块大标题 |
| banner 标题 | 48 | 67.2 | 页面 banner 标题 |
| 特殊 | 64 | 89.6 | 数据展示大数字 |

#### 第二字阶 (标题 / 展示,步长 +4/+8/+12/+16)

适用于模块超大标题、展示型文案、数字数据展示。
- 36 / 48 / 60 / 64 / 72 px …

### 3.3 字重

| Token | 值 | Windows 显示 | macOS 显示 |
| --- | --- | --- | --- |
| `--font-weight-regular` | `400` | Regular | Regular |
| `--font-weight-medium` | `500` | Regular(< 600 默认) | Medium |
| `--font-weight-semibold` | `600` | Bold | Semibold |

> **Why 400 + 600?** 在 Windows 系统中 `font-weight < 600` 默认显示为常规体 (Regular);为保证 Windows / macOS 双端显示一致,推荐只使用 `400` 和 `600` 两个字重。

### 3.4 行高

> **公式: `line-height = font-size + 8`**

行高是字号 + 8px,既满足 1.5 倍行高(14px / 16px 常用字号),又保证不同字号之间的呼吸空间一致。

| 字号 | 推荐行高 |
| --- | --- |
| 10 | 14 |
| 12 | 16.8 |
| 14 | 22 |
| 16 | 22.4 |
| 18 | 25.2 |
| 20 | 28 |
| 24 | 33.6 |
| 28 | 39.2 |
| 36 | 50.4 |
| 48 | 67.2 |
| 60 | 84 |
| 64 | 89.6 |

### 3.5 标题层级

| 层级 | 字号 / 字重 | 字体 | 颜色 |
| --- | --- | --- | --- |
| H1 (页面标题) | 60 / 600 | PingFang HK | `#030714` |
| H2 (区块标题) | 24 / 500 | PingFang SC | `#030714` |
| H3 (子区块) | 20 / 600 | PingFang SC | `#030714` |


两种模式均提供"窄工具区 (312 px)"与"宽工具区 (468 px)"两种变体。

## 6. 基础组件

### 6.1 按钮 (Button)

按钮共有 **5 种尺寸** 和 **5 种主题**,支持 4 个状态:Default / Hover (`触碰状态`)/ Pressed (`点击状态`)/ Disabled / Loading。

#### 尺寸 (Sizes)
